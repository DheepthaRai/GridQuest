# GridQuest

A classroom-friendly location game for exploring optimization on a `50x50` grid.

View the site at: [GridQuest](https://dheeptharai.github.io/GridQuest/)

## Notes

- The app is fully self-contained in `index.html`.
- No build step is required.

Built for class assignment - [MEM 410: Managerial Analytics](https://www.mccormick.northwestern.edu/engineering-management/curriculum/descriptions/410.html) by [Prof. Mike Watson](https://www.mccormick.northwestern.edu/research-faculty/directory/profiles/watson-michael.html)
